HTML Head Tag

The head Element

The head element is a container for metadata (data about data) and is placed between the HTML tag and the body tag.

The title Element

The title element defines the title of the document. It is shown in the browser's title bar or in a pages's tab.

The page title is used by search engine algorithms to decide the order when listing pages in search result.

The link Element

The link element defines the relationship between the current document and an extrenal resource. The link is most often used to link to external style sheets.

The meta Element

The meta element is typically used to specify the character set, pages description, keywords, auther of the document and viewport settings. The metadata will not be displayed on the page, but is used by browser (How to display content), by search engines for keywords and other web services.

Some of the meta tags are:

  • Define the character set used:
    • <meta charset="utf-8">
  • Defines the viewport to make your website look good on all devices:
    • <meta name="viewport" content="width=device-width">
  • Defines keywords for search engines:
    • <meta name="keyword" content="HTML, CSS, JavaScript">
  • Defines the auther of a web pages:
    • <meta name="auther" content="John Deo">
